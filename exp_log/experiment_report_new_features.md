# 横截面特征与异质因子消融实验报告

本报告系统性地总结了关于**横截面特征（Cross-Sectional Rank Features）**与**异质低相关因子（CORR/CNTP/VMA 系列）**在多因子量化选股模型中的对比回测与消融实验（Ablation Study）结果。

---

## 1. 实验设计与配置

为了保证实验的严谨性与无偏性，我们严格遵循**控制单变量**原则，维持以下基础设施不变：
- **测试样本**：486 天全样本评测（2024-05-24 ~ 2026-05-15，共 503 只股票）。
- **测试集行数损失**：0%（强无偏保证，防同分截断偏误）。
- **模型目标函数**：绝对真实收益率预测（保留跨期绝对波幅和 Beta 趋势）。
- **源基线因子（8 因子集）**：KMID, KLEN, ROC5, ROC20, ROC60, STD5, STD20, STD60。

基于此，我们设计并执行了两个阶段、共 6 组对比实验：

### 阶段一：横截面特征（Cross-Sectional Rank）探索
- **实验 1（基线 Baseline）**：仅使用 8 个基础时序绝对因子。
- **实验 2（混合特征）**：8 个基础因子 + 8 个对应的 `RANK_` 截面百分位特征（共 16 因子）。
- **实验 3（纯截面特征）**：仅使用 8 个 `RANK_` 特征，完全剔除时序绝对因子。

### 阶段二：异质低相关因子（量价/一致性）消融实验
- **实验 4（基线 + CORR5）**：5 日收盘价与 log(成交量 + 1) 的滚动相关性（第 089 号）。
- **实验 5（基线 + CNTP5）**：过去 5 日上涨天数占比（第 099 号）。
- **实验 6（基线 + VMA5）**：5 日均量 / 当日成交量（第 129 号）。

## 3. 关键发现与理论剖析

### 发现一：横截面特征在特征侧同样存在“信号稀释”现象
在上一轮对 Label 的实验中，我们证明了**截面标准化标签对模型有损害**。本轮我们在特征侧引入了自给自足的 `RANK_` 因子，实验结果表明：
1. **纯截面特征（实验 3）表现极差**：Rank IC 跌落至 0.0084，多空夏普近乎归零（0.0444），胜率跌破 50%。这强力佐证了**抛弃绝对尺度（Scale-free）会让非线性树模型丧失对跨期绝对波幅和市场趋势（Beta）的捕捉能力**。
2. **混合特征（实验 2）同样表现不佳**：虽然混合了时序绝对因子，但 Rank IC 依然从 0.0188 下滑到了 0.0114。这说明冗余的截面 rank 因子对树模型产生了“噪点污染”，导致模型分裂决策树时在“相对排名”和“绝对收益”之间产生认知冲突，导致早停点错配。

### 发现二：价量一致性因子（CNTP5）和均量比因子（VMA5）具有极佳的稳定性
在引入低相关性因子的消融实验中，我们的发现在于：
1. **均量比因子 VMA5（实验 6）稳定性拔群**：
   - 它的 Rank IC（0.0182）和多空夏普（0.1507）几乎完美逼近了最强 Baseline。
   - 更为可贵的是，它的 **IC > 0 占比（胜率）达到了 55.76%，创下所有实验的最高纪录**！且分组秩相关系数稳稳保持在最高的 **+0.7000**。
   - 这表明衡量“缩量与放量”的 `VMA5` 为模型引入了高品质的低相关量价异质信号。
2. **上涨一致性 CNTP5（实验 5）极具韧性**：
   - 保持了高达 +0.7000 的分组秩相关，这对于维持多空组合的单调性极有帮助。
3. **价量滚动相关性 CORR5（实验 4）表现较弱**：
   - 滚动 correlation 的非线性结构较为复杂，在 5 日短周期内可能噪点较多，导致模型学习难度增加，这提示我们在后续探索中可以对 CORR 考虑使用更长周期（如 CORR20 或 CORR60）来进行平滑。

---

## 4. 落地建议

1. **维持特征侧绝对因子为主的架构**：
   不建议在当前的 LightGBM 模型中大范围开启 `RANK_` 百分位特征。保留时序绝对因子（如 `ROC20`, `STD20`）有助于模型在绝对目标函数下获得最大的 Rank IC 优势。
2. **将 VMA 家族与 CNTP 家族纳为核心异质因子储备**：
   后续我们可以在 `volume.py` 中拓展更长周期的 `VMA10/20/60` 与 `CNTP10/20/60`。这些因子在线性相关性矩阵中与现有的动量/波动率因子有天然的低相关性，有助于提高模型的鲁棒性。
